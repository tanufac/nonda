# nonda
お薬服用忘れ防止アプリ

プライバシーポリシー
1. 個人情報の収集について
本アプリ「Nonda?」は、ユーザーの氏名、メールアドレス、電話番号、お薬の服用履歴などの個人情報を外部のサーバーへ収集・送信することは一切ありません。お薬の記録データは、すべてユーザーのご利用端末内（Local Storage / AsyncStorage）にのみ安全に保存されます。

2. 広告配信について
本アプリでは、第三者配信の広告サービス「Google AdMob」を利用しています。広告配信事業者は、ユーザーの興味に応じた広告を表示するためにCookieや識別子（IDFA/AAID）を使用することがあります。これらには個人を特定する情報は含まれません。AdMobのプライバシーポリシーに関しては、Google社の公式ページをご確認ください。

3. 免責事項
本アプリの利用によって生じた不利益や損害について、開発者は一切の責任を負いません。お薬の服用に関しては、必ず医師または薬剤師の指示に従ってください。

4. お問い合わせ
本アプリに関するお問い合わせは、アプリのコメント欄までご連絡ください。
